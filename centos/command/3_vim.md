# vim

#### 1.基本操作命令
#### 普通模式(Normal)
+ `a`: append current location
+ `i`: insert current location
+ `o`: open a line below
+ `A`: append after line
+ `I`: insert before line
+ `O`: open a line above
+ `y`: copy
+ `p`: parse
+ `^`: 光标移至行首
+ `$`: 光标移至行末
+ `G`: 光标移至末行
+ `1G`: 光标移至首行
+ `H`: 光标移至当前窗口首行
+ `M`: 光标移至当前窗口中间位置
#### 命令行模式(Command)
+ `% s/string1/string2/g`
